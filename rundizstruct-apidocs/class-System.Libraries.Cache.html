Class Cache

Cache class

This cache class is using Rundiz\Simple-cache.
Usage:

$Cache = new \System\Libraries\Cache();

$Cache->get('cache_id');
$Cache->save('cache_id', 'my memory');
$Cache->delete('cache_id');
$Cache->clear();

This class is for help the whole application use the same storage folder for the cached object in file system and implement some of PSR16.
