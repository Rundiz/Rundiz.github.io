# getBaseUrl( boolean $with_lang = true, boolean $trailing_slash = false )

Get base URL with multi language support.

Get base URL with multi language support.

Parameters

$with_lang
Set to true to include the language locale URI with the base URL (if configuration was set to use language via URL and/or default language is set to visible). Set to false and the language locale URI will always not included.
$trailing_slash
To return with trailing slash set this to true, otherwise set to false. If base URL is empty it will always return a single slash (/).

Returns

string

Return base URL. Example: /rundizstruct-install-path/public/[language] If you install RundizStruct in /rundizstruct-install-path and the public folder is /public. The [language] may be visible or not is up to the configuration.
