# getRoutes( )

Get the routes

Get the routes

This method must return an array of routes or even a single route to make installer works.
The required array key for each route are route_core, route_order.
For a route group, it is required route_group array key and its value is array.
For a normal route, it is required route_method, route_map_method, route_uri, route_controller, route_name.
If route_method is map then route_map_method is required and its value is array. The route_name is optional but should be unique and have your module name prefix.
For more information about route keys please read in the routes table in the database.

Example:

return array(
    array(
        'route_core' => 0,
        'route_order' => 4,
        'route_group' => array(
            '/path/group' => array(
                array(
                    '/path/1' => array('get', 'Controller:method'),
                    '/path/2' => array('map' => array('get', 'post'), 'Controller:method', 'route-name')
                )
            )
        )
    ),
    array(
        'route_core' => 0,
        'route_order' => 5,
        'route_method' => 'get',
        'route_uri' => '/path/to/page',
        'route_controller' => 'Controller:method',
        'route_name' => 'my-route-name',
    ),
    array(
        'route_core' => 0,
        'route_order' => 6,
        'route_method' => 'map',
        'route_map_method' => array('get', 'post'),
        'route_uri' => '/path/to/page2',
        'route_controller' => 'Controller:method',
        'route_name' => 'my-route-name2',
    ),
);

Returns

array
Return the array of routes, if no routes then return empty array.

# install( System\Libraries\Db\Db $Db )

Install a module.

Install a module.

This method will be call on install or add a module.
If your module contain some thing to work on install such as create table in database, you can use that command here.
If your module has nothing to work on install, you can create this method and leave it empty.

Do not insert any routes using this method! Use getRoutes() method instead.

Any installation script here if they failed, it should throw the errors to make a notice that installation was failed.
