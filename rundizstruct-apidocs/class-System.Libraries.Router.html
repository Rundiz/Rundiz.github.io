# addRoutes( array $routes )

Add routes to slim application.

Add routes to slim application.

Example usage: Simple

$Route->addRoutes(array(
    '/path' => array('any', 'Controller:method'),
));

With route name

$Route->addRoutes(array(
    '/path' => array('get', 'Controller:method', 'route-name'),
));

With map methods (custom route)

$Route->addRoutes(array(
    '/path' => array('map' => array('get', 'post'), 'Controller:method', 'route-name'),
));

With route groups

$Route->addRoutes(array(
    '/path/group' => array(
        array(
            '/path/1' => array('get', 'Controller:method'),
            '/path/2' => array('map' => array('get', 'post'), 'Controller:method', 'route-name'),
        )
    )
));

Once you call to this method, the cache will be generated. You can check with isRouteCached() method and then use addCachedRoutes() method.
