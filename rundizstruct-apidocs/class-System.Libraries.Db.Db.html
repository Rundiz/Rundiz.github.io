public string
# maybeConvertCharsetAndCollation( string $sql, string $charset = 'utf8mb4', string $collate = 'utf8mb4_unicode_ci', boolean $forceConvert = false )

Detect MySQL (or MariaDB) version and convert charset, collation to the specified values.

Detect MySQL (or MariaDB) version and convert charset, collation to the specified values.

The SQL string must have at least "CHARSET=xxx" (xxx might be utf8) or else it will have nothing to convert.
If $forceConvert is set to false then it will be check for supported charset and collation before convert it.
If the charset and collation was not support then it will return the original SQL string.

Parameters

$sql
The SQL string to be check and convert.
$charset
The MySQL charset to be convert to. Default is "utf8mb4". Do not change this due to it might affect with the whole application.
$collate
The MySQL collation to be convert to. Default is "utf8mb4_unicode_ci". Do not change this due to it might affect with the whole application.
$forceConvert
Set to true to skip check for supported charset and collation. Set to false to check it before. This is good for unit test to force and see the result.

Returns

string
Return converted or not converted if condition is not met of SQL string.

Throws

InvalidArgumentException
Throw the error on invalid argument type.
