Main page of /rds-get-languages.

Main page of /rds-get-languages.

Usage:
- Get it via ajax (json or xml content accept).
Send the request to this URL http://yourdomain.tld/rundizstruct-install-path/public/rds-get-languages?redirect_back=your_url.
With method to "GET".
Set "Accept" header with "application/json" or "application/xml" just one.
- Get it via sub request.
In your controller, use this code to make sub request. $content = $this->Modules->subRequest('get', '/rds-get-languages', 'redirect_back=your_url'); and use $content as language switch HTML.
You can add "design" to the query parameter (3rd parameters). The "design" values are "selectbox, ulist" the "ulist" is default if none specify. Example: 'redirect_back=your_url&design=selectbox'
- Get it via module execute.
In your controller, use this code to make module execute.

$request = $request->withQueryParams(['redirect_back' => 'your_url', 'design' => 'selectbox']);
   $content = $this->Modules->execute('Modules\Languages\Controllers\GetLanguages:index', [], $request);

Use $content as language switch HTML.
The "redirect_back" array key is for redirect back when language was set, "design" values are "selectbox, ulist" the "ulist" is default if none specify.
