# get( mixed $name, mixed $default = '' )

Get config value(s).

Get config value(s).

Example for get single config name:

$ConfigDb->get('site_lang_method', 'url');

Example for get multiple config name:

$names = ['site_lang_method', 'site_lang_url_default_visible'];
$defaults = ['url', 0];
$ConfigDb->get($names, $defaults);

Parameters

$name
The config name can be string (name) or array (names).
$default
The default config value if this configuration is not exists or have no value. This can be any types but if name is array then this will always be array.

Returns

mixed
Return config value. If name is array then it will always return the array with name in key or empty array in case that it is not found.
