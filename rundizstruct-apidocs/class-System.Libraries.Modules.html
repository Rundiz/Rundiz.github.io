# execute( string $module_class_method, array $args = [], Psr\Http\Message\ServerRequestInterface $request = null, Psr\Http\Message\ResponseInterface $response = null )

Manually load and execute a module.

Manually load and execute a module.

This is useful for widgetize, modular process.

To check that is this a module execute, use $_SERVER['RUNDIZSTRUCT_MODULEEXECUTE'].
Example:

if (!isset($_SERVER['RUNDIZSTRUCT_MODULEEXECUTE']) || (isset($_SERVER['RUNDIZSTRUCT_MODULEEXECUTE']) && $_SERVER['RUNDIZSTRUCT_MODULEEXECUTE'] !== true)) {
    // this is NOT module execute.
} else {
    // this is module execute.
}

Before call this method:
Make sure that the module you specified is enabled otherwise the autoload will not working
The method in the specified $module_class_method must have the same arguments as controller's method which contain $request, $response, $args.

Parameters

$module_class_method
The module class:method. The class and method should not have suffix. Example: Modules\YourModule\Controller:method. The Controller will automatically add suffix and becomes ControllerController, the method is the same it will automatically add suffix and becomes methodAction.
$args
The arguments that will be paste into third argument in the controller's method.
$request
The request interface.
$response
The response interface, to prevent body overriding please set this to null to use new \Slim\Http\Response();.

Returns

string
Return the body content of module execute. If you did not set $response as null then it will be echo automatically (body overriding), and no return works.

# subRequest( string $method, string $path, string $query = '', array $headers = [], array $cookies = [], string $bodyContent = '', Psr\Http\Message\ResponseInterface $response = null )

Make sub request to specific method and path.

Make sub request to specific method and path.

To check that is this a sub request, use $_SERVER['RUNDIZSTRUCT_SUBREQUEST'].
Example:

if (!isset($_SERVER['RUNDIZSTRUCT_SUBREQUEST']) || (isset($_SERVER['RUNDIZSTRUCT_SUBREQUEST']) && $_SERVER['RUNDIZSTRUCT_SUBREQUEST'] !== true)) {
    // this is NOT sub request.
} else {
    // this is sub request.
}

Parameters

$method
The method to make sub request. Example: any, get, post, or any Slim supported methods.
$path
Path that exists in the routes in database.
$query
The query string. To get query string use $request->getQueryParams() or $request->getParam('name').
$headers
The headers array.
$cookies
The cookies array.
$bodyContent
The body content. To get body use (string) $request->getBody().
$response
The response interface, to prevent body overriding please set this to null to use new \Slim\Http\Response();.
