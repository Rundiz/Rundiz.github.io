# getBaseUri( boolean $with_lang = true, boolean $trailing_slash = false )

Get base URI with multi language support.

Get base URI with multi language support.

Parameters

$with_lang
Set to true to include the language locale URI with the base URI (if configuration was set to use language via URI and/or default language is set to visible). Set to false and the language locale URI will always not included.
$trailing_slash
To return with trailing slash set this to true, otherwise set to false. If base URI is empty it will always return a single slash (/).

Returns

string

Return base URI. Example: /rundizstruct-install-path/public/[language] If you install RundizStruct in /rundizstruct-install-path and the public folder is /public. The [language] may be visible or not is up to the configuration.
